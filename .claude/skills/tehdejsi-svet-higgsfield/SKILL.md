# Tehdejší svět Higgsfield production

## Availability gate

Before any generation, inspect the callable tools. If the Higgsfield MCP is unavailable, do not research it, connect it, wait for it, use another generator, or create a placeholder asset. Record the deferred asset and continue every non-media task. The current deferral and asset slots live in `docs/generated-media.md`.

## Protect authenticity

Higgsfield output is decorative production material, never evidence. Do not generate historical photos, family photos, documents, newspapers, propaganda, war/famine/occupation imagery, leaders, factual maps, stars, charts, data visualizations, UI, text, logos used raw, national stereotypes, flags as branding, or third-party marks.

Real interface screenshots must be captured from the product and composited deterministically. Report text, names, dates, facts, locations, maps, sky, and UI remain code-rendered.

## Use the art direction

Follow `docs/generated-media.md`: contemporary archival collage, warm uncoated paper, clean negative space, abstract map contours, constellation plotting, restrained cut-paper depth, forest/coral/cream/ink with occasional mineral and blue-grey context. Avoid sepia nostalgia, distress, clutter, neon, glows, magical time clichés, faces, and readable generated text.

For each accepted asset, define placement, aspect ratio, text-safe area, mobile crop, negative prompt, static fallback, alt classification, and performance budget. Generate genuinely different directions; reject defects and generic results; refine only the best direction; optimize to SVG, AVIF/WebP, or WebM/MP4 as appropriate; integrate and validate in context.

Update the media manifest with prompts, negative prompts, references, variants, rejected notes, dimensions, format, bytes, crops, fallback, provenance, restrictions, and regeneration steps. Never commit credentials.
