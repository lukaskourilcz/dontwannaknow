# Tehdejší svět

Tehdejší svět pomáhá poznat prostředí, ve kterém vyrůstal člověk, na kterém vám záleží. Z ověřených dat o každodenním životě, kultuře, městech a historických událostech skládá osobní obraz dětství a dospívání — soukromě, přímo v prohlížeči.

Vite aplikace je ve složce [`dontwannaknow/`](./dontwannaknow), která zůstává Vercel Root Directory do případného externího přejmenování repozitáře.

```bash
cd dontwannaknow
npm install
npm run check
npm run dev
```

Veřejná aplikace je česká a podporuje Česko a Ukrajinu. Nepoužívá backend, uživatelské účty ani runtime AI. Osobní vstupy zůstávají v prohlížeči; sdílené zprávy používají URL fragment a standardně neobsahují jméno.

Podrobnosti jsou v [aplikačním README](./dontwannaknow/README.md), [technické dokumentaci](./DOCS.md), [design systému](./dontwannaknow/DESIGN.md) a [externím checklistu](./NEEDED.md).
